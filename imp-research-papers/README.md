# Important Research Papers in NLP

This folder contains a curated collection of important research papers in Natural Language Processing that I have read and studied.

## Purpose

This repository serves as a personal library of influential papers that have shaped the field of NLP, including foundational works, breakthrough models, and cutting-edge research.

- Each paper may include personal annotations, key takeaways, and implementation notes
- References and citations are preserved for further exploration
